Jquery is a cross-platform javascript library that is designed to simplify many scripting processes on the front end/client side.

Hi, I'm a ghost and I'll tell you about a few of the style effects in Jquery that allows you to show, hide, and toggle items in various ways.

The first 3 methods are basic functions that do their job quickly transistioning from the side.

  • .show() - This displays an element if it is hidden. 
    
	$( ".show_it" ).click(function() {
	  $( "#ghost_dude" ).show( "slow" ); 
	  // animation runs slow(600ms)
	});
  • .hide() - Hides an element. 
    
	$( ".hide_it" ).click(function() {
	  $( "#ghost_dude" ).hide( "fast" ); 
	  // animation runs fast(200ms)
	});
  • .toggle() - Toggles between hidden and displayed 
    
	$( ".toggle_it" ).click(function() {
	  $( "#ghost_dude" ).toggle( 3000 ); 
	  // completes in a very slow 3 secs (3000ms)
	});

Now to be spooky like a ghost! The next methods are like the first 3, but with now an added effect of fading in.

  • .fadeIn() - Fades the elements in. 
    
	$( ".fade_in" ).click(function() {
	  $( "#ghost_dude" ).fadeIn( "slow" );
	});
  • .fadeOut() - Fades the elements out. 
    
	$( ".fade_out" ).click(function() {
	  $( "#ghost_dude" ).fadeOut( "slow" );
	});
  • .fadeToggle() - Toggles elements with a fade effect. 
    
	$( ".fade_toggle" ).click(function() {
	  $( "#ghost_dude" ).fadeToggle( "slow" );
	});
  • .fadeTo() - Fades element to a specified opacity. 
    
	$( ".fade_low" ).click(function() {
	  $( "#ghost_dude" ).fadeTo( "slow" , 0.25);
	});
  • .fadeTo() - This example returns it to normal opacity. 
    
	$( ".fade_high" ).click(function() {
	  $( "#ghost_dude" ).fadeTo( "slow" , 1);
	});

These methods apply a sliding effect.

  • .slideDown() - Shows the element with a slide effect. 
    
	$( ".slide_down" ).click(function() {
	  $( "#ghost_dude" ).slideDown( "slow" );
	});
  • .slideUp() - Hides the element with a slide effect. 
    
	$( ".slide_up" ).click(function() {
	  $( "#ghost_dude" ).slideUp( "slow" );
	});
  • .slideToggle() - Toggles the element with a slide effect. 
    
	$( ".slide_toggle" ).click(function() {
	  $( "#ghost_dude" ).slideToggle( "slow" );
	});

Now you can easily add some style to your webpage with JQuery.
